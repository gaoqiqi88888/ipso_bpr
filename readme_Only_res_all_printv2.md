**中文版：**

这是一个MATLAB代码，专门用于评估和改进粒子群优化（IPSO）算法在图像处理中的性能。它包含两个核心模块：第一个是性能评估模块，能够对100张测试图像进行PSNR、MSE和SSIM三个关键指标的批量分析，自动执行单样本t检验并生成带显著性标记的中英文Excel统计报告，帮助科研人员量化算法改进效果



**English Version:**

This MATLAB code is designed for evaluating and improving the performance of Improved Particle Swarm Optimization (IPSO) algorithms in image processing. It consists of two core modules: The performance evaluation module enables batch analysis of 100 test images across three key metrics (PSNR, MSE, and SSIM), automatically performs one-sample t-tests, and generates bilingual Excel statistical reports with significance markers, helping researchers quantify algorithm improvements.
